XRPASCEND Mainnet Launch Steps
1) Create wallet file (seeds stay local):
   python prepare_mainnet_wallets.py
2) Fund both printed addresses with XRP on XRPL Mainnet.
3) Set environment variables from the wallet file:
   - ISSUER_SEED
   - HOT_SEED
   - XRPL_NETWORK=mainnet
4) Run:
   python launch_xrpascend.py
5) View current on-ledger token state:
   python view_xrpascend.py
6) Open explorer links from xrpascend_launch_output.json.

Important
- Never commit wallet seed files.
- Keep issuer and treasury wallets secure and separate.
